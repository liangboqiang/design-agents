This package contains a curated set of current reference files for the latest architecture stage.

How to use:
1. Compare each file with your local repository.
2. Replace matching paths where appropriate.
3. Run:
   - python -m pytest
   - python -m compileall src tests
   - git diff --check

Included focus areas:
- final naming and architecture constitutions
- runtime facade and builder
- narrowed harness contracts
- turn driver
- key runtime service boundary
- builder regression tests
